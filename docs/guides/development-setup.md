## General Setup

#### Prerequisites

- **Git**
- **Node.js** - version 14.x
- **npm**

#### Installation

Get a local copy of this repository with `git clone`

Install dependencies:

```
npm install
```

#### Running tests

Check if tests are passing

```
npm test
```

Run tests in "watch" mode

```
npm run test:watch
```

Check "scripts" section in `package.json` for more.
